🔑 Google Access Token

Sheets'e yazabilmek için geçici bir OAuth token gerekiyor.

Nasıl alınır (2 dakika):
1. OAuth 2.0 Playground'u aç
2. Sol listeden Google Sheets API v4 → https://www.googleapis.com/auth/spreadsheets seç
3. "Authorize APIs" → Google hesabınla giriş yap
4. "Exchange authorization code for tokens" → Access token'ı kopyala
5. Aşağıya yapıştır (token ~1 saat geçerli)
